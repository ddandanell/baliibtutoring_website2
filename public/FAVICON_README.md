# Favicon Setup

This directory should contain the following favicon files:

1. **favicon.ico** - 16x16 and 32x32 ICO file for browsers
2. **favicon-16x16.png** - 16x16 PNG for modern browsers
3. **favicon-32x32.png** - 32x32 PNG for modern browsers
4. **apple-touch-icon.png** - 180x180 PNG for iOS devices
5. **og-image.jpg** - 1200x630 JPG for social media sharing (Open Graph)

## Current Status
These files are referenced in `index.html` but need to be created with the actual Bali IB Tutoring branding.

## Recommended Tools
- Use a favicon generator like https://realfavicongenerator.net/
- Create an OG image at 1200x630px with the site branding
- Ensure the color scheme matches the site theme (#003E5A primary color)
